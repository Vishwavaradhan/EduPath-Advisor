1️⃣ Project Title

Edupath Advisor – One-Stop Personalized Career & Education Advisor

2️⃣ Problem Statement
A digital platform (web/app) that serves as a personalized advisor for students, professionals, and career-changers, integrating career guidance, educational pathways, and skill development recommendations into a single ecosystem.


3️⃣ Solution Overview
It uses to understand the user’s strengths, interests, market trends, and career goals, and then offers customized roadmaps (education, certifications, training, mentorship, and job opportunities).



4️⃣ Features

Personalized career assessment

Education & skill roadmap

Market insights

Mentorship & job matching

5️⃣ Tech Stack

Frontend: React, TypeScript, Vite, Tailwind CSS

Backend: Firebase (Auth, Firestore)

Hosting: Netlify 

6️⃣ Google Technologies Used

Firebase Authentication

Cloud Firestore

Firebase Security
